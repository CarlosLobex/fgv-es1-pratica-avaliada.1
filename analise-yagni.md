# Análise YAGNI

## Atributos desnecessários

- idade
- endereço
- telefone

Esses atributos não são necessários pois o sistema atual só exige nome, email e senha.

## Métodos desnecessários

- atualizar perfil
- deletar usuário
- exportar dados

Esses métodos violam YAGNI pois não são requisitos atuais.

## Explicação

O princípio YAGNI diz que não devemos implementar funcionalidades antes de serem necessárias.
